## Description

The attached program allows you to simulate real-world stock data using a list of heuristics. This stock bot will attempt to earn you the best outcome based on the stocks data you feed it. The example data we use in this project is AAPL.csv, Apple's stock history last year.
The attached report contains information pertaining to our three algorithms, a comparative analysis, and the results.
